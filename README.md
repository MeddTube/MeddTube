MeddTube
========

Like FunnyOrDie + Google Helpouts, for healthcare. So you don't, uh, die.

Key Functionalities & associated APIs:

* User model (usual login, auth, etc)

* Clean up CCR medical record, render it beautifully in new tab, and convert to JS object (BlueButton.js)

* Canned videos (Ziggeo API)

* Live videoconference (TokBox API)

* Scheduling (Makeplans.net + API)

* Eligibility checking (Eligible API)

* Provider Search vs NPI database (Bloom API)
